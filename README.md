# 学生成绩管理系统 python989

## 项目概述

学生成绩管理系统是一款基于Python语言和Flask框架，结合MySQL数据库开发的教务管理系统。该系统旨在帮助教育机构高效管理学生成绩信息。

## 技术栈

- **后端**: Python
- **框架**: Flask
- **数据库**: MySQL

## 功能模块

- 学生信息管理
- 课程信息管理
- 成绩录入与查询
- 成绩统计分析

## 系统角色

- 管理员：负责系统维护、数据管理和用户权限设置
- 教师：负责成绩的录入和查询
- 学生：查询个人成绩

## 运行环境

- Python 3.x
- Flask 1.x
- MySQL 5.x

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库
3. 克隆或下载源代码
4. 安装依赖包
5. 配置数据库
6. 运行应用
